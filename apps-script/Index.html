Trello から取り込む

  1. Trello で取り込みたいボードを開く
  2. 右上の 「…」（メニューを表示） → 「印刷、エクスポート、共有」 → 「JSON としてエクスポート」
  3. 開いた JSON のページで Ctrl+S を押して .json ファイルとして保存
  4. 下の 「ファイルを選択」 でその .json を選び、「取り込む」 を押す

※ 添付ファイルは Trello 側にあるため移行されません（リンクも引き継がれません）。それ以外（リスト・カード・期限・ラベル・チェックリスト・コメント・完了状態）は取り込まれます。

アーカイブ済みカード

このボードでアーカイブしたカードです。「リストに戻す」で元に戻せます。

設定

⏰ 期限リマインダー（メール）

期限が近い／過ぎた未完了カードを、毎朝メールでお知らせします。

🛡 自動バックアップ

全データ（DB）を別ファイルに自動コピーします。万一壊れても前の状態に戻せます（直近10個を保持）。

🔁 繰り返しカード（このボード）

毎日／毎週／毎月、自動でカードを作成します（深夜に生成）。

🔗 読み取り専用の共有（このボード）

家族や同僚にボードを「見るだけ」で共有できます。下のリンクで開けます。
※ 他の人に開いてもらうには、デプロイのアクセスを「リンクを知っている全員」にし、共有モードを有効にしてください。

🤖 自動化（このボード）

カードを特定のリストに移動したとき、自動で完了/アーカイブ/期限設定などを実行します。

🖼 ボードの背景

画像は Wikimedia Commons（自由ライセンス）から検索しています。クリックで背景に設定します。
